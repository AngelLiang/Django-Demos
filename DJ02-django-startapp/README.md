# Django创建一个应用示例

创建一个应用

    $ python manage.py startapp polls

启动

    $ cd mysite
    $ python manage.py runserver

访问

- polls: http://127.0.0.1:8000/polls/
- admin: http://127.0.0.1:8000/admin/

备注：访问 http://127.0.0.1:8000/ 可能会报错，因为没有配置路由
